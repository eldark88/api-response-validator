# Api::Response::Validator

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'api-response-validator'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install api-response-validator

## Usage

```ruby
bundle exec api-response-validator config.json
```
